# Interactive Health Sections Website

A visually stunning one-page website featuring five themed health sections with modern animations and interactive elements.

## Features

- 🎨 Beautiful SVG wave dividers between sections
- 🌓 Dark/Light theme toggle
- 🎵 Interactive sound effects
- 📱 Fully responsive design
- ♿ Accessibility features
- 🎭 3D tilt effects
- 🔄 Smooth transitions
- 🎯 Progress indicator

## Technologies Used

- HTML5
- CSS3 with modern features
- JavaScript (ES6+)
- GSAP for animations
- Howler.js for sound effects

## Sections

1. **Mental Health**
   - Interactive speech bubbles
   - Inspirational quotes
   - Floating animations

2. **Physical Activity**
   - Movement-focused content
   - Dynamic transitions

3. **Nutrition**
   - Health-focused messaging
   - Engaging visuals

4. **Sleep & Recovery**
   - Rest-themed content
   - Calming animations

5. **Social Well-being**
   - Connection-focused content
   - Interactive elements

## Setup

1. Clone the repository
```bash
git clone https://github.com/moamenalmahe/ihs-website.git
```

2. Open `index.html` in your browser
